# 文字麻将

## 联网版（推荐）

本机已安装 [Node.js](https://nodejs.org/) 时，在项目根目录执行：

```bash
npm install
npm start
```

浏览器打开终端里提示的地址（默认 **http://localhost:3000**）。

### 怎么和朋友玩

1. 其中一人打开网页，选择 **2 人 / 3 人**，点 **创建房间**。
2. 复制 **房间号**，或用 **复制链接** 把带 `?join=房间号` 的地址发给朋友。
3. 其他人用同一局域网下电脑的浏览器打开该链接；或在同一页面输入房间号点 **加入房间**。
4. 人满后服务器自动洗牌发牌，规则与本地测试版一致（吃牌、胡牌确认、8 行分类等）。

### 部署到公网（朋友不在同一 WiFi）

可把同一项目部署到任意支持 Node 的平台，例如 [Railway](https://railway.app/)、[Render](https://render.com/)、[Fly.io](https://fly.io/) 等：启动命令 `npm start`，环境变量 `PORT` 由平台注入即可。部署后把 **https 你的域名** 发给朋友，仍用「创建房间 / 加入房间」即可。

**说明：** 当前房间与对局状态在服务器内存中，进程重启后房间会消失；断线后该座位会空出，对局可能无法继续，需重新开房（朋友局可接受）。

## 项目结构

| 文件 / 目录 | 说明 |
|------------|------|
| `server.js` | Express + Socket.io，房间与同步 |
| `gameEngine.js` | 权威游戏逻辑（发牌、操作校验） |
| `public/` | 静态页面与前端脚本 |
| `public/app.js` | 浏览器 UI、与服务端同步 |
